# AppMenuMutableListOfCampo

Aplicación Android desarrollada en Kotlin con Jetpack Compose para gestionar una lista mutable de estudiantes usando el patrón Singleton.

## Características
- Menú principal con navegación a:
  - Lista de estudiantes
  - Registro de estudiante
- Lista de estudiantes mostrada con tarjetas modernas (Cards)
- Registro de nuevos estudiantes en un formulario
- Datos almacenados en un Singleton global
- Diseño moderno en modo oscuro (Material3)

## Estructura
- model/Estudiante.kt
- data/DatosEstudiantes.kt
- ui/ListaEstudiantesActivity.kt
- ui/RegistroEstudianteActivity.kt
- MainActivity.kt

## Tecnologías usadas
- Kotlin
- Jetpack Compose
- Material 3
- AndroidX
- Patrón Singleton
- mutableListOf

## Ejecución
1. Abrir el proyecto en Android Studio.
2. Sincronizar dependencias.
3. Ejecutar en emulador o dispositivo físico.
